<h1> GETTING STARTED </h1>

<h3> How to run the project locally </h3>

<h3>1. Clone the repository </h3>

```
git clone https://github.com/your-username/animeDomain.git
cd animeDomain
```

<h3> 2. Install the dependencies</h3>

```
cd animeDomain
npm install
```


<h3> 3. Run the project </h3>

```
cd animeDomain
npm run dev
```
